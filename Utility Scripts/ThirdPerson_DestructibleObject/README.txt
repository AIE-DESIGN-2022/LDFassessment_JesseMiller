1. Import the ThirdPerson_DestructibleObject.unityPackage into the Third Person Template Project.
2. Add the ObjectHealthCanvas.prefab (located in the Assets/3DGamekitLite/Prefabs folder) as a child of the object in scene that has the Damageable.cs script attached to it (Cube, enemy etc) IMPORTANT - Has to be a child of the object that has the script on it!
3. Adjust the child object (named "Slider") position to desired location (move the Slider not the Canvas!)
4. Adjust Max Hit Points on the Damageable script in inspector.
